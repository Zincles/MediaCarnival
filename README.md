## 功能？

它是一个类似于 Emby 的媒体库. 你可以认为目前它还是 emby 的拙劣仿制品. (它还是在制品) <br>

## 部署？

目前请不要尝试本机部署. 即使目前侥幸部署成功了,你大概也会被无数愚蠢的漏洞给气到心肺停止吧.<br>
你需要同时运行 Django 后端与 vite(quasar)前端, 安装完所有需要的依赖后, 才可以完整启动该项目. <br>
